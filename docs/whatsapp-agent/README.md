# WhatsApp Sales Agent - Integration

Hier legt Max die Erklaerungs-MD vom anderen Agent ab, die beschreibt wie der
WhatsApp Sales Agent nach Formular-Absendung den Interessenten kontaktiert.

## Zweck dieses Ordners
- Dokumentation des Sales Agents (Funktionsweise, Trigger, Flows)
- Payload/Datenstruktur die das Formular an den Agent schicken muss
- Webhook-URLs, API-Keys, n8n Flow-Referenzen
- Edge Cases / Error Handling

## Ziel
Das Reiter-Formular muss Live und mit dem WhatsApp Sales Agent verbunden werden,
damit nach Absenden automatisch der WhatsApp-Kontakt erfolgt.
