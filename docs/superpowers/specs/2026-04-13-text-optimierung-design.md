# Text-Optimierung Reiter Immobilien LP

## Ziel

Psychologisch optimierte Texte für alle Sektionen der Landing Page. Fokus auf hochqualifizierte Verkäufer-Leads (keine Vergleicher/Neugierige). Universelle Texte für beide Traffic-Quellen (Meta-Ads + Flyer).

## Psychologische Hebel

| Hebel | Wo eingesetzt |
|-------|--------------|
| Verlustaversion | Hero, Homestaging (15% mehr) |
| Sozialer Beweis | Trust-Bar mit Google-Logo, Reviews mit konkreter Anzahl |
| Autorität | 30+ Jahre, Verena persönlich, lokale Marktkenntnis |
| Differenzierung | "Persönlich statt Portal" durchgängig |
| Niedrige Hemmschwelle | "Kostenlos", "Unverbindlich", "3 kurze Schritte" |
| Qualifizierung | Sprache setzt Verkaufsabsicht voraus |
